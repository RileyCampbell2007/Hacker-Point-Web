User Scripts

Userscripts are small programs or scripts that modify the behavior or appearance of websites. They are typically written in JavaScript and are executed by web browsers. Users can install a browser extension, such as Tampermonkey or Greasemonkey, which provides a runtime environment for running userscripts. Once installed, the userscript can automatically run on specific websites, allowing users to customize their browsing experience by adding new features, removing ads, or altering website functionality.

Bookmarklets

Bookmarklets are small JavaScript programs that can be saved as bookmarks in web browsers. When clicked, they execute a specific action on the current web page. To run a bookmarklet, you simply click on the bookmark that contains the JavaScript code, and it gets executed within the context of the open web page, allowing you to modify, enhance, or interact with its content in various ways.
